Bike Stunt Game

Description

Bike Stunt Game is a mobile game where players perform physics-based bike stunts across challenging tracks. The game uses Bike Physics Controller for realistic bike movement and balance mechanics.

Players can perform flips, jumps, and tricks while managing bike physics. Features include multiple levels, progressive difficulty, realistic controls, and stunt challenges.

This project highlights Unity physics integration, bike controller scripting, 3D environment creation, and mobile game optimization.
